## Arquivo de entrega ao cliente

O ZIP entregue após a compra deve conter as 5 imagens em alta resolução.
Nome do arquivo:

```
respiro-de-garopaba-wallpapers.zip
```

Ideal: resolução 1440×2560 ou maior para o ZIP de entrega, mesmo que as imagens exibidas na página sejam otimizadas para web.
